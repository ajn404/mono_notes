## 目录

## 概念

> programming:表格法

> 同分治算法一样，是通过组合子问题的解来解决原问题。

> 动态规划算法通常用于求解最优化问题。

> 应用于子问题重叠的情况


## 步骤

- 1. 分析最优解的性质，并刻画其结构特征
- 2. 递归地定义最优解的值
- 3. 计算最优解的值，通常采用自底向上的方法
- 4. 利用计算出的信息构造一个最优解
