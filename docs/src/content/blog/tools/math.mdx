## 添加插件

```bash
pnpm --filter @notes/docs i remark-math rehype-katex -r
```

```js
import remarkToc from "remark-toc";
import remarkMath from "remark-math";
import { defineConfig } from "astro/config";
//***
export default defineConfig({

    // ***
    markdown: {
        remarkPlugins: [
            remarkMath,
        ],
        rehypePlugins: [
            rehypeKatex,
             
         ],
})
```
